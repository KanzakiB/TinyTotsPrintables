# TinyTots Printables

An e-commerce platform offering printable learning materials for early childhood education, designed for children aged 0-8 years.


## About The Project

TinyTots Printables is an educational e-commerce website that provides a diverse range of printable resources for teachers, parents, and caregivers. The platform offers engaging materials that promote cognitive, motor, and social skill development in young learners. By providing affordable, high-quality educational tools, TinyTots Printables aims to support early childhood education both inside the classroom and at home.


## Features

- Printable worksheets, activity sheets, coloring pages, flashcards, and puzzles
- User registration and login
- Product browsing and purchasing
- Downloadable digital resources
- In-store printing service
- Delivery options for Pasig residents


## Built With

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL / MariaDB
- Server: XAMPP / Apache


## Installation

Step 1: Install and run XAMPP
Start Apache and MySQL in the XAMPP Control Panel.

Step 2: Create the database
Go to localhost/phpmyadmin and create a new database name **tinytotsdb**.

Step 3: Import the SQL file
In phpMyAdmin, select your database, click the Import tab, choose the SQL file, and click Go.

Step 4: Move project files
Copy the project folder to C:\xampp\htdocs\


## Usage

| Page | Local URL |
|------|-----------|
| **Homepage** | `http://localhost/TinyTotsPrintables/frontend/homepage.php` |
